Описание задания
Необходимо провести отладку приложения для парсинга публичных
вакансий с сайта Selectel. Проект реализован с помощью фреймворка
Fast API:
- осуществляется парсинг вакансий каждые 5 минут;
- информация сохраняется в PostgreSQL базе данных;
- предоставляется CRUD API для операций с вакансиями (чтение,
добавление, изменение, удаление).

В приложении намеренно допущены баги, которые не позволяют
корректно выполнять работу. Необходимо запустить приложение
локально и провести отладку (исправить баги), чтобы приложение
работало исправно. Умышленно внедрено 8 багов разной степени
сложности

Критерии успешно выполненной работы
1. Успешно запустить приложение (без ошибок)
2. Убедиться, что парсинг вакансий выполняется корректно (без
падений)
3. Обеспечить работу всех CRUD-операций
5. Убедиться, что фоновая задача парсинга работает по заявленному
расписанию.
Формат сдачи
Необходимо написать отчёт о проделанной работе в произвольном
формате:
- какие проблемы были выявлены и как удалось их решить;
- какие баги были обнаружены и как можно их исправить.

`None`
* Код после: `if price_str: item.price = int(price_str)`
* Причина: Не была выполнена проверка на `None` перед
конвертацией.
